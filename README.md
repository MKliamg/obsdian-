## 🚀 快速开始

### 1. 下载 Obsidian
前往 [Obsidian 官网](https://obsidian.md/) 下载适合你操作系统的版本。

### 2. 创建知识库
打开 Obsidian，选择"创建新库"，选择一个文件夹作为你的笔记存储位置。

### 3. 开始记笔记
按 `Ctrl/Cmd + N` 创建新笔记，使用 `[[]]` 创建链接，开始构建你的知识网络！

### 4. 探索插件
进入设置 → 第三方插件 → 关闭安全模式 → 浏览社区插件。
