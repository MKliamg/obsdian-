# 示例文件目录

本目录包含各种 Obsidian 使用示例，可以直接复制到你的知识库中使用。

## 📁 目录结构

```
examples/
├── README.md              # 本文件
├── dataview-queries.md    # Dataview 查询示例
└── sample-notes/          # 示例笔记
```

## 📝 示例说明

### Dataview 查询示例

[dataview-queries.md](dataview-queries.md) 包含常用的 Dataview 查询代码，可以直接复制使用。

## 🚀 如何使用

1. 浏览示例文件
2. 复制需要的代码或内容
3. 粘贴到你的 Obsidian 笔记中
4. 根据需要修改参数

## 💡 提示

- 示例代码需要安装对应的插件才能运行
- 根据你的文件夹结构调整路径
- 参考示例创建自己的模板
