## 必备插件

这些插件能显著提升 Obsidian 的使用体验，推荐新手优先安装。

| 插件 | 功能 | 推荐度 |
|------|------|--------|
| **Dataview** | 将笔记作为数据库查询 | ⭐⭐⭐⭐⭐ |
| **Templater** | 高级动态模板系统 | ⭐⭐⭐⭐⭐ |
| **Calendar** | 日历视图管理日记 | ⭐⭐⭐⭐⭐ |
| **Tasks** | 全库任务追踪管理 | ⭐⭐⭐⭐⭐ |
| **QuickAdd** | 快速添加笔记和执行命令 | ⭐⭐⭐⭐ |

### Dataview

```
功能：从笔记中查询和汇总数据
文档：https://blacksmithgu.github.io/obsidian-dataview/
教程：[[../advanced-techniques/01-dataview]]
```

**典型用途**：
- 创建自动更新的笔记列表
- 追踪项目进度
- 汇总任务和待办
- 生成统计报告

### Templater

```
功能：创建动态模板，自动填充日期、执行脚本
文档：https://silentvoid13.github.io/Templater/
教程：[[../advanced-techniques/02-templater]]
```

**典型用途**：
- 自动插入当前日期
- 弹窗获取用户输入
- 文件自动重命名
- 执行自定义脚本

### Calendar

```
功能：侧边栏日历视图，点击日期快速跳转
GitHub：https://github.com/liamcain/obsidian-calendar-plugin
```

**典型用途**：
- 可视化每日笔记
- 快速创建和跳转日记
- 查看周报入口

### Tasks

```
功能：跨笔记任务管理和查询
文档：https://publish.obsidian.md/tasks/
```

**典型用途**：
- 收集全库所有任务
- 按日期、优先级筛选
- 任务重复设置
- 完成日期追踪

## 编辑增强

提升编辑效率的插件。

| 插件 | 功能 | 说明 |
|------|------|------|
| **Outliner** | 大纲增强 | 类似 Workflowy 的操作 |
| **Advanced Tables** | 表格编辑增强 | Excel 式表格操作 |
| **Editing Toolbar** | 编辑工具栏 | 可视化格式按钮 |
| **Paste URL into selection** | URL 粘贴优化 | 选中文字粘贴为链接 |
| **Auto Link Title** | 自动获取链接标题 | 粘贴 URL 自动获取标题 |
| **Linter** | 格式规范化 | 自动格式化 Markdown |
| **Various Complements** | 自动补全 | 文本自动补全 |

### Outliner

```markdown
# 功能
- 使用 Tab/Shift+Tab 缩进
- 拖拽重排列表项
- 折叠/展开列表
- 快速移动列表项
```

### Advanced Tables

```markdown
# 功能
- Tab 键导航单元格
- 自动对齐表格
- 添加/删除行列
- 排序表格数据
```

## 插件安装建议

### 新手阶段

```
1. 先使用核心插件熟悉 Obsidian
2. 添加 Calendar 和 Tasks
3. 学习 Templater 创建模板
4. 探索 Dataview 做查询
```

### 进阶阶段

```
1. 添加 QuickAdd 提升效率
2. 使用 Kanban 做项目管理
3. 配置 Git 进行备份
4. 探索 Smart Connections
```

### 注意事项

- ⚠️ 不要一次安装太多插件
- ⚠️ 定期检查更新
- ⚠️ 注意性能影响
- ⚠️ 备份后再尝试新插件
