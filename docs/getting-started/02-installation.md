## 下载安装

### Windows

1. 访问 [Obsidian 官网](https://obsidian.md/)
2. 点击 **Download** 按钮
3. 下载 `.exe` 安装文件
4. 双击运行安装程序
5. 按照提示完成安装

> **提示**：也可以下载便携版（Portable），无需安装，可放在 U 盘中使用。

### macOS

1. 访问 [Obsidian 官网](https://obsidian.md/)
2. 点击 **Download** 按钮
3. 下载 `.dmg` 文件
4. 打开 `.dmg` 文件
5. 将 Obsidian 拖动到 Applications 文件夹

或使用 Homebrew：
```bash
brew install --cask obsidian
```

### Linux

**Debian/Ubuntu (AppImage)**：
```bash
# 下载 AppImage
wget https://github.com/obsidianmd/obsidian-releases/releases/download/v1.5.3/Obsidian-1.5.3.AppImage

# 添加执行权限
chmod +x Obsidian-*.AppImage

# 运行
./Obsidian-*.AppImage
```

**Flatpak**：
```bash
flatpak install flathub md.obsidian.Obsidian
```

**Snap**：
```bash
sudo snap install obsidian --classic
```

### 移动端

| 平台 | 下载方式 |
|------|----------|
| iOS | App Store 搜索 "Obsidian" |
| Android | Google Play / APK 直接下载 |

## 创建知识库

### 什么是知识库（Vault）

知识库就是一个**文件夹**，Obsidian 将其作为工作空间，所有笔记都存储在这个文件夹中。

### 创建步骤

1. 首次打开 Obsidian，你会看到启动界面
2. 选择 **创建新库（Create new vault）**
3. 输入知识库名称（例如：`我的笔记`）
4. 选择存储位置

> **最佳实践**：
> - 选择云盘文件夹（如 iCloud、OneDrive）可实现多设备同步
> - 使用简短、有意义的名称
> - 避免使用中文路径中的特殊字符

### 文件夹结构建议

```
我的笔记/
├── 00-收件箱/         # 快速记录，稍后整理
├── 01-日记/           # 每日笔记
├── 02-项目/           # 进行中的项目
├── 03-领域/           # 长期关注的领域
├── 04-资源/           # 参考资料
├── 05-归档/           # 已完成的内容
├── 模板/              # 笔记模板
└── 附件/              # 图片、文件等
```

## 界面介绍

### 主界面布局

```
┌─────────────────────────────────────────────────────┐
│  ← → 导航  │  标签页  │  标题栏                     │
├──────┬──────────────────────────────────┬───────────┤
│      │                                  │           │
│ 侧   │                                  │  右       │
│ 边   │        编 辑 区 域               │  侧       │
│ 栏   │                                  │  边       │
│      │                                  │  栏       │
│      │                                  │           │
├──────┴──────────────────────────────────┴───────────┤
│  状态栏                                             │
└─────────────────────────────────────────────────────┘
```

### 核心区域

| 区域 | 功能 |
|------|------|
| **左侧边栏** | 文件浏览器、搜索、书签 |
| **编辑区域** | 笔记编辑/预览 |
| **右侧边栏** | 反向链接、大纲、标签 |
| **状态栏** | 字数统计、编辑模式等 |

### 常用面板

- **文件浏览器**：管理笔记和文件夹
- **搜索**：全文搜索（Ctrl/Cmd + Shift + F）
- **书签**：收藏常用笔记
- **大纲**：当前笔记的标题结构
- **反向链接**：哪些笔记链接到当前笔记
- **图视图**：可视化笔记关系

## 基础设置推荐

进入设置：点击左下角 ⚙️ 或按 `Ctrl/Cmd + ,`

### 编辑器设置

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| 默认编辑模式 | 实时预览 | 所见即所得，适合新手 |
| 拼写检查 | 开启 | 帮助发现拼写错误 |
| 显示行号 | 开启 | 便于定位 |
| 缩进使用Tab | 关闭 | 使用空格更通用 |
| Tab 缩进大小 | 2 或 4 | 个人偏好 |

### 文件与链接

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| 新建笔记存放位置 | 当前文件夹 | 或指定文件夹 |
| 附件默认存放位置 | 指定文件夹 | 如 `附件/` |
| 链接格式 | 相对路径 | 便于笔记迁移 |
| 使用 Wiki 链接 | 开启 | `[[链接]]` 格式 |

### 外观设置

```markdown
# 推荐设置
- 字体大小：16-18px（根据屏幕调整）
- 主题：可选深色/浅色，或下载社区主题
- 显示行内标题：开启
- 缩减标签栏：关闭（多标签页更方便）
```

### 核心插件

建议开启的核心插件：

| 插件 | 功能 |
|------|------|
| ✅ 反向链接 | 查看链接到当前笔记的其他笔记 |
| ✅ 大纲 | 显示笔记的标题结构 |
| ✅ 标签面板 | 管理所有标签 |
| ✅ 日记 | 每日笔记功能 |
| ✅ 模板 | 笔记模板系统 |
| ✅ 搜索 | 全文搜索 |
| ✅ 图视图 | 知识图谱可视化 |
| ✅ 命令面板 | 快速执行命令 |
| ✅ 星标 | 收藏重要笔记 |

## 快捷键配置

### 必备快捷键

| 操作 | Windows/Linux | macOS |
|------|---------------|-------|
| 新建笔记 | `Ctrl + N` | `Cmd + N` |
| 打开快速切换 | `Ctrl + O` | `Cmd + O` |
| 打开命令面板 | `Ctrl + P` | `Cmd + P` |
| 搜索 | `Ctrl + Shift + F` | `Cmd + Shift + F` |
| 打开设置 | `Ctrl + ,` | `Cmd + ,` |
| 切换编辑/预览 | `Ctrl + E` | `Cmd + E` |
| 添加链接 | `Ctrl + K` | `Cmd + K` |
| 粗体 | `Ctrl + B` | `Cmd + B` |
| 斜体 | `Ctrl + I` | `Cmd + I` |
| 关闭当前标签 | `Ctrl + W` | `Cmd + W` |

### 自定义快捷键

1. 进入 设置 → 快捷键
2. 搜索想要设置的命令
3. 点击 `+` 添加快捷键
4. 按下组合键完成设置

> **推荐自定义**：
> - 插入模板：`Ctrl/Cmd + T`
> - 今日日记：`Ctrl/Cmd + D`
> - 打开图视图：`Ctrl/Cmd + G`

## 常见问题

### Q1: 知识库放在哪里比较好？

**推荐选择**：
1. **云盘目录**（iCloud、OneDrive、Dropbox）- 自动同步
2. **本地固定位置** - 配合手动备份
3. **Git 仓库** - 版本控制 + 备份

**避免**：
- 系统临时目录
- 程序安装目录
- 没有写入权限的位置

### Q2: 如何备份我的知识库？

```bash
# 方法1：定期复制整个文件夹
cp -r 我的笔记/ 我的笔记_backup_$(date +%Y%m%d)/

# 方法2：使用 Git
cd 我的笔记
git init
git add .
git commit -m "备份"
```

### Q3: 可以打开多个知识库吗？

可以！
- 点击左下角的库名称
- 选择 "打开其他库"
- 新库会在新窗口打开

### Q4: 设置可以同步到其他设备吗？

设置保存在 `.obsidian` 文件夹中。如果你的知识库在云盘中同步，设置也会一起同步。
